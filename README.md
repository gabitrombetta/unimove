# UniMove

UniMove é uma plataforma de caronas universitárias que conecta estudantes que compartilham trajetos semelhantes, facilitando a organização de viagens de forma simples e segura.

## Objetivo

O projeto está sendo desenvolvido para permitir que estudantes possam:

- Publicar caronas disponíveis;
- Buscar viagens por origem e destino;
- Solicitar vagas em viagens;
- Gerenciar solicitações de carona.
